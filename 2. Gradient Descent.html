Gradient descent algorithm¶

Gradient descent is an iterative optimization algorithm for finding the minimize any function. To find a local minimum of a function using gradient descent, one takes steps proportional to the negative of the gradient of the function at the current point. If instead, one takes steps proportional to the positive of the gradient, one approaches a local maximum of that function. the procedure is then known as gradient ascent.

Gradient descent is also known as steepest descent.

Some important terms-¶

1. Hypothesis function-¶

This is the function, which is used in unsupervised learning algorithm (linear regression, logistic regression, etc) to predict the output for some unknown data. Or we can say, the function that best fits the data is called the hypothesis function.

eg. h(X) = theta[0] + theta[1]*X¶

here theta is coefficient of vector X and is a vector itself.

In the above diagram we could have many hypothesis functions. But all the green lines do not seem to fit the data well.

But the red line seems to fit data in the best possible way. The line that fits the data in best way, is decided by minimizing the cost function.

Cost function-¶

Cost function on minimization provides value of vector theta.

J(theta) = (sum(h(X)-Y)** 2)/2m¶

m-> no of examples

Steps involved-¶

Here m denotes the number of examples in the training set.

  1. Calcuation of the hypothesis h = X * theta
  2. Calculation the cost = ((h-x)^2)/2m Some othe method can also be used to calculate the cost, in this case we are using mean square error function.
  3. Calculation of the gradient = X' * loss / m
  4. Updation of the parameters theta = theta - alpha * gradient
  5. The gardient descent will stop when the gradient becomes 0 as at this point we get local minima.

def gradientDescent(x, y, theta, alpha, m, numIterations):
    xT = x.transpose()
    
    #Loop from 0  to number of iteration
    for i in range(0, numIterations):
        
        #defining hypothesis function and loss
        hypothesis = np.dot(x, theta)
        loss = hypothesis - y
        
        #Defining cost function 
        cost = np.sum(loss ** 2) / (2 * m)
        
        #printing  no of iteration and cost for that iteration
        print("Iteration %d | Cost: %f" % (i, cost))
        # avg gradient per example
        gradient = np.dot(xT, loss) / m
        # simultaneous updation of theta
        theta = theta - alpha * gradient
    return theta


def dataGeneration(Points, bias, variance):
    x = np.zeros(shape=(Points, 2))
    y = np.zeros(shape=Points)
    
    for i in range(0, Points):
        # bias feature
        x[i][0] = 1
        x[i][1] = i
        y[i] = (i + bias) + random.uniform(0, 1) * variance
    return x, y

# generating 50 points with a bias of 20 and 10 variance as a bit of noise
x, y = dataGeneration(50, 20, 10)
m, n = np.shape(x)
numIterations= 50
alpha = 0.0003
theta = np.ones(n)
theta = gradientDescent(x, y, theta, alpha, m, numIterations)
