Linear Regression¶

What is regression?¶

Regression is a technique which identifies relation between dependent and independent variable. example: relation between height and weight of men.

What is Linear Regression¶

Linear regression is a supervised algorithem. It identifies a linear relationship between one dependent and some independent variable. Based on this relation it can predict output value for unknown independent variables. ie. Univariate linear regression can identify relationship between number of houses and population in a particular area from the givien data and then for some new data of houses it can predict the population.

How the relationship is identified?¶

To find the realtionship between variable, linear regression creates a hypothesis function-

y = h(x) = theta(1) +theta(2)x¶

where y-> population x -> No of houses

As shown in the above diagram, we can take various values of theta but we need h(x) such that it best fits the data. To find the most optimum values of theta we try to minimize the cost function and the most common cost function is the mean square error function-

J(theta) = 1/(2*m)( (y-theta )**2)¶

Then for new value of independent variable the dependent variable is predicted using the hypothesis. The following diagram shows how the population is predicted for new instances.
